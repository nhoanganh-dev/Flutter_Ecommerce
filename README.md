## Thông tin chung
- **Ngôn ngữ lập trình**: Dart

- **Nền tảng hỗ trợ**: Android, Web (Chrome)
- **Trạng thái dự án**: Đã chạy `flutter clean` để làm sạch các file build trước khi nộp.
